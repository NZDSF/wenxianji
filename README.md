# 问仙纪
魔前一叩三千年，欲问苍天何为仙。  缘起 这是一款纯粹的修仙世界。没有华丽特效，没有氪金诱导，只有一颗求道之心。  当年深夜独坐，突然想起小时候玩过的文字MUD游戏——那种纯粹靠想象构建的仙侠世界，反而比现在的3D大作更令人沉迷。于是动手做了这款WAP游戏，用最简单的代码，重现最初的那份感动。  为何开源 修仙路漫漫，一人难成道。开源，是希望更多志同道合之人能参与进来，添砖加瓦，让这个世界更加完善。代码如功法，本该广传天下，让每个有缘人都能在此基础上创造属于自己的仙侠世界。  何为道友 道友，非泛泛之交，乃同道之友。你我皆是红尘过客，于茫茫代码中寻一方净土，在虚拟世界里圆一个仙侠梦。既然同行修仙路，便是缘分。  道友，请随我来，共证大道！
